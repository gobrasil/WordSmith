Important!

WordSmith is biult on the back of OOP and MVC.

WordSmith provides an interface for developers to organize and refine their code into single use classes by extending the classes provided in the framework

If for whatever reason you are not familliar with the following topics consider using the crucible to get yourself started;


  • Object Oriented Programming
  • Namespaces
  • Class Inheritance
  • MVC (Model View Controller)

There is no standard folder structure for using the framework, simply write your own classes and extend the ones provided.

for beginer developers seeking a complete soltuion The Crucible is the best way to get started with WordSmith.
