Installation

installing WordSmith is simple, the only dependency on your web server is a functional WordPress installation and you’re good to go.

There are two main ways to install WordSmith; as a composer package or using The Crucible.

Via Composer

Installing via Composer is simple, just type to following command in and you’re good to go.

composer require nk2580/wordsmith

Composer will add the library in it’s vendor folder and from there you can extend the provided classes then include them into your theme.

WARNING

installing the composer package requires you to create your own classes, folders & autoloaders, if youre not comfortable with adding these things into a theme, or would prefer not to, please use The Crucible instead.

The Crucible

The Crucible is the fully fledged WordSmith Framework package which provides a clean structure and tonnes of great features to kick start your App or Site.
