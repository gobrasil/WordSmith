Attributes
hook WordPress Action hook where this code will exectue
callback the internal method you create for this action
priority the priority of this action at the nominated hook
args number of arguments the callback accepts
Methods
__construct DO NOT OVERRIDE
this method gathers the variables for the callAction method
callAction DO NOT OVERRIDE
this method add the action into WordPress

Actions in wordSmith are Quite Simmilar to those in vanilla WordPress, the only difference is that they appear in class form

When adding a new action in create a new file and place it somewhere in your theme or plugin.next ensure the class extends the supplied wordsmith class. from there you supply your class argumets mich like your moethod arguments orginally and then write your action method.

it's important to remeber to instantiate the newly created class, otherwise the object will not be loaded and the code wont run.
